    LayerMaskFilter

    Used to filter GameObjects by their LayerMask, we can use an inclusive layer so the GameObject need at least one layer from it to be accepted.
    We can also exlude a GameObject when it has at least one layer set inside the ExcludedLayerMask.
    Or we can use a combination of both for example you may have objects using the layer 'Enemy' and others that use 'Enemy' and 'Boss'.
    And if you want to detect 'Enemy' only, you set the bit in the Inclusive mask, and set the 'Boss' bit inside the exclusive layer mask.
