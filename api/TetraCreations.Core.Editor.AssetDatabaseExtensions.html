    Class AssetDatabaseExtensions

    Provides few generic static method to simplify getting a specific asset or a list of assets from one or multiple folders.
    Relative folder path mean this one is valid : "Assets/Game/Data/Sword.asset", but not this one : "C:\Unity\MyProject\Assets\Game\Data\Sword.asset".
    We can search using the asset name or a predicate function.
    If an asset is not found, it returns null or an empty list depending on the method called.
